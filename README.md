# Desafio Frontend

Bem-vindo ao Desafio Frontend da Asterisco! Este teste tem como objetivo avaliar suas habilidades de desenvolvimento web usando [Astro](https://astro.build/), [TailwindCSS](https://tailwindcss.com/) e [Vue](https://vuejs.org/). Sua tarefa é construir um catálogo de produtos que busca e exibe dados da Fake Store API.

## Requisitos

1. **Catálogo de Produtos**: Crie um catálogo de produtos que consuma a [Fake Store API](https://fakestoreapi.com/).
2. **Barra Lateral (Sidebar) com Categorias**: O catálogo deve ter uma lista na barra lateral contendo todas as categorias.
3. **Lista de Produtos**: Exiba os produtos em formato de grade (grid).
4. **Paginação**: Implemente paginação para a lista de produtos.
5. **Filtragem por Categoria**: Ao clicar em um link de categoria, exiba apenas os produtos daquela categoria.
6. **Exibição do Produto**: Cada produto na lista deve exibir o título, a imagem e a avaliação (rating).
7. **Página de Detalhes do Produto**: Crie uma página para cada produto que exiba todos os seus detalhes, incluindo aqueles mostrados na listagem de produtos.

## Stack (Tecnologias)

* **Astro**: Use o framework Astro para gerar um site estático.
* **TailwindCSS**: Use TailwindCSS para estilização.
* **Vue**: Use Vue.js para criar componentes.

## Entrega

* Faça um *fork* do repositório que contém este README.
* Forneça um link para o repositório público contendo seu código.
* Certifique-se de que seu repositório inclua um README com instruções sobre como rodar o projeto localmente.

## Critérios de Avaliação

* **Qualidade do Código**: Código limpo, legível e bem documentado.
* **Funcionalidade**: Todos os recursos listados funcionam conforme o esperado.
* **UI/UX**: A interface do usuário é intuitiva e visualmente atraente.
* **Performance**: Busca de dados (*data fetching*) e renderização eficientes.
* **Conclusão**: Adesão aos requisitos e à stack especificada.
